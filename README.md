# sql-data-warehouse-project
A Modern Data Warehouse model following the ETL process with Medallion Architecture.
This is my first project and I have learned how data operations are performed in the Bronze, Silver and Gold Layer.

==================================================================

Project Overview:

1. Dataware house-
          a. Bronze Layer - loading data from csv files into crm and erp tables
          b. Silver Layer - cleaning bad data, handling incorrect date time formant, performing data standardization and normalization
          c. Gold Layer - creating report and business ready data that gives meaning to our analysis
2. ETL Pipeline - Extract, Transform and Load process where we extract data from each table, transform data as per our needs and load the new data into new table
3. Star Schema - Data Modelling where the Dimension table (descriptive tables) are linked to the Facts Table (numeric table)
